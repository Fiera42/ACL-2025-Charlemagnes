# BACKLOG - itération n°2
## Objectifs
- Rattraper dettes techniques
- Avancer sur les fonctionnalités avancées

## Cas d'utilisation (CU)
### CU1 : Gestion des tags
- **Création de tags** : Ajouter la possibilité de créer des tags pour les rendez-vous.
- **Attribution de tags aux rendez-vous** : Permettre d'attribuer des tags aux rendez-vous.
- **Suppression de tags** : Permettre de supprimer des tags.
- **Modification de tags** : Permettre de modifier les tags existants.
- **Filtrage par tags** : Ajouter la possibilité de filtrer les rendez-vous par tags.
- **Recherche sur les tags** : Améliorer la barre de recherche pour permettre la recherche par tags.

### CU2 : Fonctionnalités avancées des rendez-vous
- **Rendez-vous récurrents** : Ajouter la possibilité de créer des rendez-vous récurrents.
- **Lister les rendez-vous avec des conflits** : Afficher une liste des rendez-vous qui ont des conflits d'horaires.

### CU3 : Profil utilisateur
- **Consulter son profil utilisateur** : Ajouter une page pour consulter et modifier les informations de son profil utilisateur.
